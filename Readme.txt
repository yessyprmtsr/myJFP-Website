WEBSITE myJFP adalah sebuah website dimana melayani pemesanan kurir untuk pengantaran barang secara online. Disini terdapat dua user yaitu admin dan customer. 
Untuk customer dapat melihat jenis service apa saja yang disediakan, lalu dapat memesan barang, melakukan pembayaran, dan melihat riwayat pemesanan.
Untuk admin dapat memanage semua data dari mulai majemen user,transaksi,barang,dan tracking dari barang yang dikirim kurir.
